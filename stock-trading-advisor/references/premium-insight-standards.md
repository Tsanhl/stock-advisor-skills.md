# Premium Insight Standards

Use this standard to produce research with executive/advisory value rather than generic summary.

## 1. Differentiate From Consensus

For each key point:
- State the consensus/base narrative.
- State what your analysis adds or challenges.
- Show the evidence that supports the difference.

## 2. Quantify Consequences

Translate findings into magnitude and timing:
- Market size, cost impact, probability ranges, sensitivities, or timeline windows.
- If exact values are unavailable, provide bounded estimates and assumptions.

## 3. Decision Utility

For each major insight, answer:
- Who should act?
- What action is supported?
- What is the expected upside/downside?
- What signal would invalidate this recommendation?

## 4. Scenario Discipline

Provide at least three scenarios where relevant:
- Base case
- Upside case
- Downside case

For each scenario, include likely triggers and a confidence label.

## 5. Evidence Integrity

- Use primary data where possible.
- Mark all assumptions clearly.
- Keep facts, interpretation, and speculation explicitly separated.
